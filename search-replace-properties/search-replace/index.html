# Search Levels

  • Expressions – search and replace within Expressions
  • Layers – search and rename Layers
  • Project Items – search and rename Project Items

TIP

When the search level is set to Expressions, only selected Properties are queried. If you wish to query all expressions in your composition, Search & Select all properties with expressions, by using the wildcard selector * and setting its Has Expression argument first.

# Fields

You can stack multiple queries in one run. Multiline text is supported for each field.

Input fields

  • Search – the text to search for
  • Replace – the text to replace the search text with
  • + – add a new search query to the stack
  • - – remove the search query from the stack

# Arguments

  • Math Case – respect lower- and uppercase during the query
  • Match Word – match complete words only, no partials

# Buttons

  • Dry Run – run the Search/Replace queries, and print results in a separate window. No changes are done in the process
  • Apply – run and apply the Search/Replace stack

# Shortcuts

  • CTRL+ENTER + [active text-field] – add a new query to the stack
  • ALT+CLICK + [Dry Run] – clear all results
  • ALT+CLICK + [Apply] – delete and clear all Search/Replace fields in the stack
