# Search & Replace Properties

# Search & Select

# Search Levels

The Search Levels dropdown menu, lets you select the level at which the query shall start.

  • Property-Level – search and select layer properties
  • Layer-Level – start by querying layer names, and proceed with layer properties. Matching layers and properties are selected
  • Project Items – search and select items in Project Pane

# Wildcard Selector

Use the character * to search and select all properties that meet the chriteria defined by the Arguments.

# Separators

> a hierarchical parent/child indicator

, separates each property that is to be searched and selected for

; marks the end of a query. You can chain as many queries as you like

# Arguments

Can Exp. – only Properties that can have expressions are selected

Has Exp. – only Properties that have expressions assigned are selected

Match Case – only Properties which's name match the exact spelling are selected

Strict – only properties that are a direct descendant of the parent group are selected (works for Shape Layers only)

TIP

In order to search for specific Text animators using strict mode, the following hidden groups have to be considered in the search hierarchy:

  • Text Animator groups have to be preceded with the hidden Animators group name
  • Range Selector groups have to be preceded width the hidden Selectors group name
  • Transofrm Properties have to be preceded width the hidden Properties group name

# Examples

#' Example query #1

scale,rotation

Search for all properties named Scale and Rotation

Result:

├── Tranform
│   ├── Scale
│   ├── Rotation

 
 

# Example query #2

group>scale,rotation

Search for all properties named Scale and Rotation, which are a child of a Property Group named Group

Result:

├── Group 1
│   ├── Tranform
│   │   ├── Scale
│   │   ├── Rotation
├── Group 2
│   ├── Tranform
│   │   ├── Scale
│   │   ├── Rotation


 
 


 
 

# Example query #3

Query: group 1>group>scale,rotation

  1. search for property group named group 1
  2. search for all nested property groups, where group is a match of the property groups' name
  3. select all scale and rotation properties within the nested property group

Result:

├── Icon Group
│   ├── Shape Group 1
│   │   ├── Tranform
│   │   │   ├── Scale
│   │   │   ├── Rotation
│   ├── Shape Group 2
│   │   ├── Tranform
│   │   │   ├── Scale
│   │   │   ├── Rotation



 
 


 
 

# Example query #4

Example query with Strict mode activated.

Query: Icon Group>Shape Group 2>scale,rotation

  1. search for the property group named Icon Group
  2. search for the nested property group named Shape Group 2
  3. select all scale and rotation properties within Shape Group 2

Result:

├── Icon Group
│   ├── Shape Group 2
│   │   ├── Tranform
│   │   │   ├── Scale
│   │   │   ├── Rotation



 
 

# Search & Replace

# Search Levels

  • Expressions – search and replace within Expressions
  • Layers – search and rename Layers
  • Project Items – search and rename Project Items

TIP

When the search level is set to Expressions, only selected Properties are queried. If you wish to query all expressions in your composition, Search & Select all properties with expressions, by using the wildcard selector * and setting its Has Expression argument first.

# Fields

You can stack multiple queries in one run. Multiline text is supported for each field.

Input fields

  • Search – the text to search for
  • Replace – the text to replace the search text with
  • + – add a new search query to the stack
  • - – remove the search query from the stack

# Arguments

  • Math Case – respect lower- and uppercase during the query
  • Match Word – match complete words only, no partials

# Buttons

  • Dry Run – run the Search/Replace queries, and print results in a separate window. No changes are done in the process
  • Apply – run and apply the Search/Replace stack

# Shortcuts

  • CTRL+ENTER + [active text-field] – add a new query to the stack
  • ALT+CLICK + [Dry Run] – clear all results
  • ALT+CLICK + [Apply] – delete and clear all Search/Replace fields in the stack
